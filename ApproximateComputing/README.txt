This folder contains the various files used to accomplish approximate computing on Gem5 simulation tool. This folder is the master folder which contains all of the 
subfolders, which describes the specific part of the project the underlying files work. The subfolders have READMEs that show the actual folder in the project that the corresponding files
are soft linked. This allows all of the files to be centrally located in a concise manner and have soft links to the specific location in the project.
